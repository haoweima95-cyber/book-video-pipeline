# Text Segment：按句末标点拆为逐行

读取任意文案 txt，按句末标点（`。！？；……`）将段落拆分为逐行，原地覆盖原文件。

```bash
python main.py 文案.txt --sentences
```

处理逻辑：
1. 按句末标点（`。！？；……`）初拆
2. 合并短句：相邻句累计 ≤60 字就合并为一行
3. 拆长句：超过 60 字的行在逗号/顿号/分号/冒号处切分
4. 结语"感谢你的观看，我们下本书再见。"保护不参与合并/拆分
5. 原地覆盖原文件

## 所需权限

- `Read` — 读取 txt
- `Write` — 写入 txt
- `Bash` — 运行 python
