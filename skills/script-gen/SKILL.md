# 文稿生成

给定书名 → 读种子范本 → 查知识库风格 → 写文案 → 逐行换行 → 建文件夹。

## 步骤 0：读取种子范本（必须，不可跳过）

读取 `<book-video-pipeline>/seed/samples/` 下全部 `.txt` 文案。

这些是高赞读书口播文案的**质量基准线**。后续生成的文案在：
- 内容密度（~2000+ 字，非 1500 字敷衍）
- 叙事层次（钩子→概念堆叠→策略→哲学收束，非列举式）
- 概念颗粒度（有命名、有比喻、有数据，非泛泛而谈）

上不得低于种子范本。读完后你的脑中应该有清晰的"好文案长什么样"的画面。

## 步骤 1：查询风格并撰写文案（querying-the-wiki）

调用 `querying-the-wiki` skill，查询知识库中读书口播文案的共性风格。**结合步骤 0 的种子范本和 wiki 返回的风格规范**，为指定书籍撰写短视频文案。

直接输出到项目文件夹内（pipeline Step 1 已建好）：`<工作目录>/<书名>/<书名>口播文案.txt`。

## 步骤 2：按句换行（text-sentence）

```bash
python <text-sentence-dir>/main.py "<工作目录>/<书名>/<书名>口播文案.txt" --sentences
```

原地覆盖，将段落拆为逐行文本，供后续合并。

## 步骤 3：合并短行 + 拆分长行（text-sentence --merge）

```bash
python <text-sentence-dir>/main.py "<工作目录>/<书名>/<书名>口播文案.txt" --merge
```

合并 ≤60 字的相邻短行，拆分 >60 字的长行。结语不参与合并/拆分。原地覆盖。

⚠️ **到此为止，不要建 `script/`，不要拆单句。** 用户需要先审阅修改文案。

> 告知用户：文案已生成，请审阅修改。修改确认后我会先**总结本次改动**，再等待下一步指令。

## 步骤 4：生成视频简介

参照 `d:\自媒体\` 下已有项目的 `视频简介与分段.txt` 格式，生成到 `<工作目录>/<书名>/视频简介与分段.txt`：

```
标题：《书名》钩子句

简介：作者+一句话定位，核心问题链，收束句。

#标签：#书名 #作者 #核心关键词
```

格式要求见下方"视频简介与分段"写作规范。

## 最终输出

```
<书名>/
├── <书名>口播文案.txt       ← 逐行文案全文（可编辑）
├── 视频简介与分段.txt        ← 标题+简介+标签
├── images/                  ← 待放生图
└── images_video/            ← 待放视频片段
```

> 后续流水线：
> 1. `text-split --split` → `script/N.txt`
> 2. `txt-to-prompt` → `_prompt.txt`，`sd-image-gen` → `images/`
> 3. `img-to-video` → `images_video/`
> 4. `voice-subtitle` → `voice_output/`
> 5. `video-compose` → `segments/`
> 6. `segment-concat` → `final.mp4`

## 步骤 5：整理写作经验

⚠️ **用户确认"修改好了"后，必须立即自动调用 `writing-feedback` skill。** 无需等待用户说"整理经验"或手动触发 `/writing-feedback`——这是脚本生成的强制收尾步骤。

> ⚠️ writing-feedback 对比时，用户修改后的文件必须用 `Bash cat` 直接读取，不可用 Read（可能返回 text-sentence 工具原地覆盖前的缓存）。

## 写作规范（基于用户编辑反馈持续更新）

### 标点

- **禁止使用破折号（——）**。口播文案中破折号读不出来，全部用逗号（，）或"那就是""也就是"替代。
- 引号仅用于强调特定概念（如"目标悖论""踏脚石"），日常用词不加引号。
- **引号并列项之间加顿号分隔。** 多个引号包裹的短并列项连排时，项与项之间加顿号（、），口播时停顿更清晰。
  - ❌ `你父母可能对你说"不许哭""有什么好怕的""再哭我就走了"`
  - ✅ `你父母可能对你说"不许哭"、"有什么好怕的"、"再哭我就走了"`
- **引号并列项之间加顿号分隔。** 多个引号包裹的短并列项连排时，项与项之间加顿号（、），口播时停顿更清晰。
  - ❌ `你父母可能对你说"不许哭""有什么好怕的""再哭我就走了"`
  - ✅ `你父母可能对你说"不许哭"、"有什么好怕的"、"再哭我就走了"`

### 断句：合并

- **不要过度拆分**。语义紧密的邻句合并为一行，2-3 句合并是合理的。
- 常见合并场景：

| 场景 | 说明 | 示例 |
|---|---|---|
| 铺垫+展开 | 引出句与紧随的内容合并 | ❌ `还有一个故事更让我震撼。` / `1513年，西班牙探险家…` → ✅ `还有一个故事更让我震撼。1513年，西班牙探险家…` |
| 设问+答案 | 自问自答合并 | ❌ `结果你猜怎么破的？` / `他们发现有一扇小门…` → ✅ `结果你猜怎么破的？他们发现有一扇小门…` |
| 因果链 | 动作与结果合并 | ❌ `几千人的军队就从这扇门里涌了进去。` / `一千年的拜占庭帝国，终结于…` → ✅ `几千人的军队就从这扇门里涌了进去。一千年的拜占庭帝国，终结于…` |
| 转折 | 让步与转折合并 | ❌ `你可能觉得这只是一本讲历史偶然性的书。` / `但它真正扎心的地方不在这里。` → ✅ `你可能觉得这只是一本讲历史偶然性的书。但它真正扎心的地方不在这里。` |
| 正反对举 | 正说与反说合并 | ❌ `那个瞬间出现了，你就是群星中的一颗。` / `你错过了，你就是尘埃。` → ✅ `那个瞬间出现了，你抓住了，你就是群星中的一颗。你错过了，你就是尘埃。` |
| 总起+第一条 | 总起句与第一条论点合并 | ❌ `这本书给了我三个启发。` / `第一，认得出关键时刻。` → ✅ `这本书给了我三个启发。第一，要认得出关键时刻。` |
| 论点+论证 | 分论点与论证合并 | ❌ `第三，拥抱意外。` / `我们总想掌控一切…` → ✅ `第三，拥抱意外。我们总想掌控一切，但茨威格…` |
| 连续动作 | 事件起因与后续动作合并 | ❌ `有一天上头让他写一首军歌。` / `他熬了一夜，写出了《马赛曲》。` → ✅ `有一天上头让他写一首军歌。他熬了一夜，写出了《马赛曲》。` |
| 铺垫+升华 | 铺垫与升华句合并 | ❌ `我们每个人都在经历自己的历史。` / `而你此刻正在做的选择…` → ✅ `我们每个人都在经历自己的历史。而你此刻正在做的选择，就是你生命中的群星时刻…` |
| 发现+追问 | 研究发现后立刻追问延伸 | ❌ `他发现在员工的工作动力中，薪酬和物质激励只占了大概15%。` / `剩下的85%是什么？` → ✅ `他发现在员工的工作动力中，薪酬和物质激励只占了大概15%。那剩下的85%是什么？` |
| 事实+来源 | 抛出数字/事实后紧跟佐证 | ❌ `钱只能解决15%的问题。` / `这个数字不是我编的，是考夫曼基于几十年的研究和实践得出的结论。` → ✅ `钱只能解决15%的问题。这个数字不是我编的，是考夫曼基于几十年的研究和实践得出的结论。` |
| 铺垫+设问 | 陈述铺垫后紧跟设问句（"结果你猜发生了什么？"）合并 | ❌ `管理层觉得这套系统完美无缺，所有行为都能量化。` / `结果你猜发生了什么？` → ✅ `管理层觉得这套系统完美无缺，所有行为都能量化。结果你猜发生了什么？` |
| 设问+答案+命名 | 设问→答案→概念命名三句连环推进时合并 | ❌ `而更可怕的是什么呢？` / `这种网络隔离直接导致了机会的不平等。` / `杰克逊把它叫做社会固化。` → ✅ `而更可怕的是什么呢？这种网络隔离直接导致了机会的不平等。杰克逊把它叫做社会固化。` |
| 并列举例 | 相同句式的2-3个例子合并为一行，一气呵成 | ❌ `孩子摔倒了哭，你说"不疼不疼"。` / `孩子害怕黑暗，你说"勇敢一点"。` / `孩子说我不喜欢吃的…` → ✅ `孩子摔倒了哭，你说"不疼不疼"。孩子害怕黑暗，你说"勇敢一点"。孩子说我不喜欢吃的，你说"必须吃"。` |
| 连续反问 | 同一主语下的连续反问句合并为一行 | ❌ `他怎么可能有自信？` / `怎么可能建立边界？` / `怎么可能喊停？` → ✅ `他怎么可能有自信？怎么可能在人际关系里建立边界？怎么可能在别人伤害他的时候喊停？` |
| 短排比合并 | 结构对称的短排比句合并为一行 | ❌ `你焦虑，他就紧张。` / `你能接纳自己，他才能接纳自己。` / … → ✅ `你焦虑，他就紧张。你能接纳自己，他才能接纳自己。你能承认错误，他才敢承认错误。你活成了什么样子，他就长成什么样子。` |
| 命名+核心论断 | 概念命名紧跟其解释定义句合并 | ❌ `你的愤怒…下意识逃避。` / `佩里管这个叫"来自过去的幽灵"。` → ✅ `你的愤怒，其实是下意识逃避。佩里管这个东西叫做"来自过去的幽灵"。` |

### 断句：拆分

- **超长的冒号解释句应拆分。** 冒号后如果堆了多个并列项（举例、排比），拆为"引入 → 核心结论 → 举例"三行，让观众听得清每一层。
  - ❌ `茨威格告诉你一个扎心的真相：改变人类命运的，往往不是那些宏大的、计划好的事情，而是一个微不足道的瞬间，一个意外，一个冲动，甚至一个错误。`
  - ✅
    ```
    茨威格告诉你一个扎心的真相：
    改变人类命运的，往往不是那些宏大的、计划好的事情，而是一个微不足道的瞬间。
    比如一个意外，一个冲动，甚至一个错误。
    ```
- **"不是…而是…"长句应在转折点拆分。** 先否定后肯定，在"而是"处断开，制造停顿张力。
  - ❌ `他说人类最深层的恐惧，不是死亡本身，而是害怕自己的生命毫无意义，害怕离开这个世界的时候没有留下任何痕迹。`
  - ✅
    ```
    他说人类最深层的恐惧，不是死亡本身。
    而是害怕自己的生命毫无意义，害怕离开这个世界的时候没有留下任何痕迹。
    ```
- **长因果/递进解释句应拆分。** 原因与结果、递进链在逻辑交界处断开。
  - ❌ `考夫曼说，这比没有使命还糟糕，因为它制造了一种叫做"价值观虚伪"的东西，你说的和做的不一样，员工会立刻嗅到那种虚伪，然后彻底关掉意义感的开关。`
  - ✅
    ```
    考夫曼说，这比没有使命还糟糕，因为它制造了一种叫做"价值观虚伪"的东西。
    你说的和做的不一样，员工会立刻嗅到那种虚伪，然后彻底关掉意义感的开关。
    ```
- **冒号列举句拆分。** 冒号后是一串并列动作/条目时，冒号前独立成行，并列项另起一行。
  - ❌ `他提出了一个承诺管理的五步循环：怎么做出承诺，怎么要求承诺，怎么获得承诺，怎么重新商谈承诺，怎么追究责任。`
  - ✅
    ```
    他提出了一个承诺管理的五步循环：
    怎么做出承诺，怎么要求承诺，怎么获得承诺，怎么重新商谈承诺，怎么追究责任。
    ```
- **观点与比喻拆分。** 抽象观点后接生活化比喻时，在比喻处断开，让听众先消化观点再接收画面。
  - ❌ `穆勒建议把指标从绩效考核中剥离出来，把它变成一个纯粹的信息工具，就像你车上的仪表盘，油量低了是提醒你加油，不是用来扣你工资的。`
  - ✅
    ```
    穆勒建议把指标从绩效考核中剥离出来，把它变成一个纯粹的信息工具。
    就像你车上的仪表盘，油量低了是提醒你加油，不是用来扣你工资的。
    ```
- **总起与排比拆分。** 定性/总起句与后续的举例排比拆为独立两行，给听众一拍消化时间。
  - ❌ `我们生活在一个数据崇拜的时代，你的运动步数、你的阅读量、你的粉丝数、你的收入、你的资产，每一天都有无数个数字在定义你是谁。`
  - ✅
    ```
    我们生活在一个数据崇拜的时代。
    你的运动步数、你的阅读量、你的粉丝数、你的收入、你的资产，每一天都有无数个数字在定义你是谁。
    ```
- **作弊行为与系统反应拆分。** 描述指标造假的"行为→系统反应"链时，在行为与欺骗结果之间断开，加"这样"点破因果。
  - ❌ `还有一些医院更聪明，在急诊室旁边临时搭了帐篷，把快超时的病人推到帐篷里，系统显示已经接诊了，实际上医生根本没来。`
  - ✅
    ```
    还有一些医院更聪明，在急诊室旁边临时搭了帐篷，把快超时的病人推到帐篷里。
    这样系统显示已经接诊了，但实际上医生根本没来。
    ```
- **长排比引入句在连接词处拆分。** 三项以上"不管叫A、叫B、还是叫C"的排比引入句，在最后一项"还是叫C"后断开，让后续分论点独占一行，避免前半句过长吞掉核心信息。
  - ❌ `不管它叫计划经济、叫人工智能调控、还是叫算法分配，只要有人告诉你"把所有决策交给我们来做会更好"，你就应该想起哈耶克的这本书。`
  - ✅
    ```
    不管它叫计划经济、叫人工智能调控、还是叫算法分配，
    只要有人告诉你"把所有决策交给我们来做会更好"，你就应该想起哈耶克的这本书。
    ```

### 结构

- **冒号前缀上收拆分。** 冒号引出句（如"作者举了一个真实的案例："）与上文语义紧密相连时，将冒号前缀合并到上一行，冒号后的内容独立成行。前缀跟着语境走，内容独占一行，口播节奏更清晰。
  - ❌
    ```
    致富需要冒险、需要乐观、需要往前冲。
    但守富需要的是谦卑、是恐惧、是时刻准备承认自己可能是错的。
    作者举了一个真实的案例：一个人如果每年赚30%，连续赚了10年，但第11年亏了50%…
    ```
  - ✅
    ```
    致富需要冒险、需要乐观、需要往前冲。但守富需要的是谦卑、是恐惧、是时刻准备承认自己可能是错的。作者举了一个真实的案例：
    一个人如果每年赚30%，连续赚了10年，但第11年亏了50%…
    ```

- **钩子问题必须有回扣。** 开头抛出的对比/设问，正文中必须找到合适位置直接回答，不能只提问题不管答案。
  - ❌ 开头问"为什么有些人睡6小时却精神抖擞"，正文讲完全部内容也没回扣。
  - ✅ 在讲解睡眠纺锤波时回扣："这就回到了开头那个问题：为什么有些人睡六个小时就能精力充沛？答案就在这里。"

### 语气

- 避免绝对化表述。加"可能""往往""几乎"等软化词。
  - ❌ `恰恰是唯一能到达的路`
  - ✅ `可能恰恰是唯一能到达的路`
- 年龄、时间等涉及具体数字的表述留余地："二十多岁"而非"二十岁"。
- **第一人称感受转第三人称陈述。** "这本书给了我"→"他列出了"，更客观可信，避免口播者个人色彩过浓。
  - ❌ `这本书给了我四个特别重要的启发。`
  - ✅ `他列出了四个特别重要的启发。`
- **钩子段可用第一人称直述，不必处处归因于书。** 开头钩子需要快速击中观众，"这本书会告诉你"不如"我会告诉你"有力。书的权威信息后置到作者背书段再展开。
  - ❌ `但今天我聊的这本书会告诉你一个让你睡不着觉的真相：`
  - ✅ `但今天我会告诉你一个扎心的真相：`
- **加"但"字强化转折。** "其实不是"→"但其实不是"，口播中半拍停顿后"但"字给出更清晰的逻辑信号。
  - ❌ `你以为员工最想要的是钱，其实不是。`
  - ✅ `你以为员工最想要的是钱，但其实不是。`
- **论断加归属词。** 关键结论前加"他认为""作者说"等，明确这是书中观点而非口播者个人断言。
  - ❌ `指标不是解决问题的工具，指标本身就是问题。`
  - ✅ `他认为：指标不是解决问题的工具，指标就是问题本身。`

### 衔接

- 段落之间用口语化连接词过渡："那就是""也同样""换句话说"。
  - ❌ `撞上了一个发现——设定目标`
  - ✅ `撞上了一个发现，那就是设定目标`
- **强化对比时加"而"字。** 单用"是"→"而是"，增加对比张力。
  - ❌ `最后这段话不是在讲管理学理论了，是在讲一种看待工作和人生的方式。`
  - ✅ `最后这段话不是在讲管理学理论了，而是在讲一种看待工作和人生的方式。`
- **加"这样"点破因果。** 行为与（欺骗性）结果之间加"这样"，让观众跟上逻辑跳跃。
  - ❌ `把快超时的病人推到帐篷里，系统显示已经接诊了`
  - ✅ `把快超时的病人推到帐篷里。这样系统显示已经接诊了`
- **并列论证加"此外"过渡。** 两个并列的危害/论证之间（如情绪失控→记忆力衰退），加"此外"让逻辑层次更清晰。
- **加"不过"引出让步转题。** 从上一段的结论过渡到反面注意事项时，句首加"不过"让转折更平滑，不突兀。
  - ❌ `接纳感受不等于纵容行为。孩子可以生气，但不能打人。`
  - ✅ `不过接纳感受不等于纵容行为。孩子可以生气，但不能打人。`
  - ❌ 情绪失控段与记忆力段之间无过渡词
  - ✅ `此外，你的记忆力也会被严重打击。`

### 用词

- **"要"换"得到"，动词更有结果感。** "全都要"太模糊抽象，"全都得到"让观众脑中有具体画面，口播更清晰。
  - ❌ `你永远不可能全都要。`
  - ✅ `你永远不可能全都得到。`
- **去学术后缀。** 口播中专业术语不宜带学术后缀（如"量""率""化"），去掉之后更自然，观众不需要在脑中"翻译"术语。
  - ❌ `理性人考虑边际量`
  - ✅ `理性人考虑边际`
- **感性词换真诚词。** "诚实"带冷感和审视意味，"真诚"更温和可亲。描述作者/书中态度时优先用温情词替换冷感词。
  - ❌ `曼昆在书的结尾说了一段非常诚实的话。`
  - ✅ `曼昆在书的结尾说了一段非常真诚的话。`
- 选择更口语、更日常的表达，避免书面化或抽象词。
  - ❌ `最后这段话不是在讲历史了，是在讲一种活法。`
  - ✅ `最后这段话不是在讲历史了，是在讲一种生活态度。`
- 动词前加"要"字让口播节奏更自然。
  - ❌ `第一，认得出关键时刻。`
  - ✅ `第一，要认得出关键时刻。`
- **英文术语换中文口语词。** 口播避免中英夹杂，听众反应不过来。
  - ❌ `领导者最大的敌人不是竞争对手，而是自己的ego。`
  - ✅ `领导者最大的敌人不是竞争对手，而是自己的自尊心。`
- **单字动词扩展为双字，节奏更稳。** "换"→"转换"、"变"→"改变"，双音节在口播中更自然。
  - ❌ `那个问题一换，你的整个工作状态就变了。`
  - ✅ `当那个问题一转换，你的整个工作状态就开始变了。`
- **动词前加"带来""开始"等铺垫词。** 让动作有过程感而非突兀发生。
  - ❌ `我能给这个世界什么`
  - ✅ `我能给这个世界带来什么`
  - ❌ `你的整个工作状态就变了`
  - ✅ `你的整个工作状态就开始变了`
- **去极端化后果描述。** 避免过于尖锐、戏剧化的后果，改用委婉的负向评价。
  - ❌ `病人却死了`
  - ✅ `医术却不见得特别高明`
- **去戏剧化身体感受。** 日常生活场景不用暴力/夸张比喻（如"像被人打了一顿"），用中性表述代替。
  - ❌ `醒来那一刻却像被人打了一顿，浑身沉重`
  - ✅ `起床的时候却浑身沉重`
- **去口语隐喻。** 行业/圈内口语化的隐喻换为直白标准表述，口播面向大众。
  - ❌ `指标都绿了`
  - ✅ `指标都合格了`
- **外部定义转主体选择。** "X定义你是谁"改为"你用X来定义自己"，被动变主动，更有主体性。
  - ❌ `当指标开始告诉你你是谁`
  - ✅ `当你开始靠指标来定义自己是谁`
- **避用 TTS 多音字陷阱词。** "少即是多"的"少"常被 CosyVoice 误读为 shào（少年）而非 shǎo（多少），且易与前面的"第三"粘连成"第三少"。直接换说法，如"少就是多"或拆开表述。
  - ❌ `第三，少即是多。`
  - ✅ `第三，考核指标越少反而越好。`
- **加"也"增强并列节奏。** 排比/并列结构中加"也"字，口播节奏更流畅，强化对等关系。
  - ❌ `认识的人比你牛`
  - ✅ `认识的人也比你牛`
- **加"却"字强化并列结构内部转折。** 枚举平行项时，在关键项前加"却"字，在不另起从句的前提下制造对比感，比"但"更轻、更融合。
  - ❌ `理财成功与否，跟你的智商几乎没关系，跟你如何处理情绪、如何看待时间、如何控制贪婪，关系巨大。`
  - ✅ `理财成功与否，跟你的智商几乎没关系，却跟你如何处理情绪、如何看待时间、如何控制贪婪，关系巨大。`

- **加"还是"强化排比节奏。** 三项排比的最后一项前加"还是"，口播中给出更清晰的节奏信号，避免"叫A、叫B、叫C"的急促堆叠。
  - ❌ `不管它叫A、叫B、叫C，只要…`
  - ✅ `不管它叫A、叫B、还是叫C，只要…`
- **加"是在"软化拟人陈述。** 拟人化的"X说"改为"X是在说"，加"是在"让语气更自然、不那么生硬。
  - ❌ `当价格下跌了，它说"这个过剩了"`
  - ✅ `当价格下跌了，它是在说"这个过剩了"`
- **"不只是…而是…"转"不仅仅是…还有…"。** 上下文中已有多个"不是…而是…"句式时，换用"不仅仅是…还有…"避免句式重复过密，同时语气更柔和。
  - ❌ `你失去的不只是自由，而是人之为人的尊严`
  - ✅ `你失去的不仅仅是自由，还有你的尊严`
- **数字格式统一为阿拉伯数字。** 年份、具体数量、日期等数字统一用阿拉伯数字（1955年、1470个、1月1日），口播时更直观，少占视觉宽度。小数字叙事中的"一个""两次"等保留中文。
  - ❌ `盖茨一九五五年出生` / `一千四百七十个儿童` / `二〇〇六年`
  - ✅ `盖茨1955年出生` / `1470个儿童` / `2006年`
- **经典引文转口语化表述。** 圣经、古文等书面引文必须换成现代大白话，听众没有时间在脑子里"翻译"文言。
  - ❌ `凡是有的，还要加给他，叫他有余；没有的，连他所有的也要夺过来。`
  - ✅ `已有优势的人，会不断获得更多且愈发富足；而一无所有的人，却连仅有的部分也终将失去。`
- **专业术语去书面化。** 即使是准确的专业词，如果太书面也换掉。如"计算机终端"→"电脑"、"机时"→"机"，口播受众不需要术语精确性，需要的是瞬时理解。
  - ❌ `花钱买了一台计算机终端，让学生上机编程` / `蹭机时`
  - ✅ `花钱买了一台电脑，让学生上机编程` / `蹭机`
- **"那就是""也就是"可换"比如"等更轻的口语词。** 上下文已有多个"那就是"时，换"比如""就像"等更轻的连接词，避免重复。
  - ❌ `那就是你的出生月份、你的家庭背景…`
  - ✅ `比如你的出生年月、你的家庭背景…`
- **代词"他"转"作者"避免指代不清。** 长段落中"他"可能指代书里出现过的任何人物，转"作者"明确指代。
- **"我"转"自己"更客观。** 叙述中"我怎么不关注他"转"自己怎么会不关注孩子"，把个人立场转为普遍陈述，更有说服力。
  - ❌ `很多父母觉得我每天24小时都在为孩子忙，我怎么不关注他了？`
  - ✅ `很多父母觉得自己24小时都在为孩子忙，怎么会不关注孩子？`
  - ❌ `他列出了三条。`
  - ✅ `作者列出了三条。`

### 精简

- 删掉不影响语义的口水词："你现在""什么都不为""你现在的"。
  - ❌ `每周留半天，什么都不为，就是瞎折腾`
  - ✅ `每周留半天，不设立目标，就是瞎折腾`
- **删过渡性设问。** "什么意思？"这类过渡设问可以删，下一句直接展开逻辑更紧凑。
  - ❌ `考夫曼说，每一个组织本质上都是一个"不朽工程"。什么意思？他说人类最深层的恐惧…`
  - ✅ `考夫曼说，每一个组织本质上都是一个"不朽工程"。他说人类最深层的恐惧…`
- **删多余"但"字。** 上下文已有明显转折时，"但其实"→"其实"，避免"但"字过密造成口播疲劳。
- **删段首"而"字。** 新段或承接段开头的"而"字往往多余，删掉更干净利落。
  - ❌ `而这本书要给你的，不是一堆育儿技巧…`
  - ✅ `这本书要给你的，不是一堆育儿技巧…`
- **删具体渲染例。** 核心论点已说明清楚时，不必再用极端具体例子展开。如"他说扮小狗，你就趴下"已在前文"他说玩什么，你就陪他玩什么"中覆盖，删掉不损信息量。
  - ❌ `聊到这里，你可能觉得，这不就是在讲人脉的重要性吗？但其实没那么简单。`
  - ✅ `聊到这里，你可能觉得，这不就是在讲人脉的重要性吗？其实没那么简单。`
- **删冗余设问拆解。** 一句设问后用另一种说法重复问一遍，是口播废词。直接用一个最精简的版本覆盖，更紧凑。
  - ❌ `这个政策好不好？它的出发点是好的还是坏的？`
  - ✅ `这个政策是好是坏？`
- **删重复主语。** 自我对话/自我安慰句式中重复主语（"我就是…我就是…"），删掉第二个保留一个即可，口播更流畅。
  - ❌ `我就是体质不好、我就是需要更多睡眠。`
  - ✅ `我就是体质不好、需要更多的睡眠。`
- **钩子场景概括化。** 开头钩子不宜用过于详细的场景描写拖慢节奏，用一句直接概括更有效。
  - ❌ `而有些同事只睡六个小时，进公司的时候双眼放光，你跟他说话的时候，你甚至能感觉到他的反应比你还快半拍。`
  - ✅ `而有些人只睡六个小时，却能立刻从疲惫的状态中恢复到精力充沛。`

### 结语

- **每篇口播文案必须以固定结语结尾。** 最后一行必须是：
  - ✅ `感谢你的观看，我们下本书再见。`
- 结语独占一行（即 text-split 后会是独立的最后一行），不与其他内容合并。
- 结语不参与内容拆分，由 text-split 自动合并到倒数第二句末尾。

### 视频简介与分段

- **标题格式：《书名》前置。** 书名放在标题最前面，后面接钩子句，用空格分隔。
  - ❌ `工资再高也留不住人？因为你漏掉了85%的动力｜《意义革命》`
  - ✅ `《意义革命》工资再高也留不住人？因为你漏掉了85%的动力`
- **标题钩子避免抽象比喻，用直接问题句。** 比喻式钩子需要观众先理解隐喻才能 get 到，而直接的问题句不依赖任何背景知识，一眼就能击中痛点。钩子应直指书的核心问题本身，而非用诗意包装。
  - ❌ `《通往奴役之路》通往奴役的路，都是善意铺成的` — 需要知道"善意铺路→奴役"这个隐喻才能理解
  - ✅ `《通往奴役之路》自由是如何被慢慢剥夺的` — 不读书也知道在讲什么，直接击中痛点
